创意介绍

💡 想解决什么问题

在信息爆炸的时代，知识工作者面临三大核心困境：知识碎片化——笔记散落在各处，难以形成体系；思维线性化——传统文档无法展现知识点之间的复杂关联；创作割裂化——记录想法、整理资料、深度编辑需要在多个工具间切换。

🎯 产品形态

AstroKnot 是一款基于 Electron + Three.js 的桌面端知识管理应用。它将每个知识点抽象为 3D 空间中的「星节点」，节点间的关联以螺旋连线呈现，配合星云粒子效果，让整个知识网络如宇宙星图般直观可见。用户可以在 3D 视图中自由探索、在 2D 思维导图中整理结构，双击任意节点即可打开功能完整的富文本编辑器进行深度创作。

✨ 核心创新点

  • 三维空间知识映射 — 打破传统二维笔记的局限，用 3D 星图形式展现知识的立体结构
  • 全栈式创作环境 — 集成富文本、表格、公式、代码、音频、视频等 12+ 种内容类型于单一节点内
  • AI 深度融合 — 内置 Chat/Agent 双模式 AI 对话，支持项目上下文感知的工具调用
  • 视图自由切换 — 3D 星图 / 2D 思维导图无缝切换，适应不同思考场景

目标用户及痛点

🎓

学生 / 研究者

使用场景：课程笔记整理、论文文献管理、课题研究梳理

当前痛点：笔记分散在不同 App 中，复习时难以建立跨学科的知识关联；文献和笔记之间缺少直观的链接关系。

💼

知识工作者

使用场景：产品需求文档、技术方案沉淀、行业研究分析

当前痛点：传统文档是线性的，无法体现业务模块之间的依赖关系；需要同时使用 Notion（记录）+ XMind（梳理）+ Word（输出），效率低下。

👨‍💻

开发者 / 技术写作者

使用场景：技术博客写作、开源项目文档、API 设计文档

当前痛点：需要频繁插入代码块、数学公式、架构图，现有工具对混合内容的支持不够友好；缺乏能同时承载文字、代码、图表的创作环境。

🧠

创意工作者

使用场景：剧本大纲构思、游戏世界观设定、创意灵感捕捉

当前痛点：想法是网状的，但工具是线性的；需要一种能像「思维导图」一样发散、又像「文档」一样深入的表达方式。

核心功能

融合 3D 可视化、富文本编辑、AI 对话的下一代知识管理平台

🌌

3D 知识网络

基于 Three.js 渲染，螺旋连线、星云粒子、节点光环，支持旋转/缩放/平移，让知识网络如星图般璀璨。

✏️

富文本编辑器

基于 TinyMCE 7，支持字体/段落/样式、图片自由放置、形状、文本框、文件链接、分栏等专业级编辑能力。

📊

表格 & 公式编辑器

基于 Univer 的完整电子表格编辑能力 + LaTeX 公式编辑器，常用符号面板，支持拖拽缩放。

💻

代码块编辑

基于 Monaco Editor，支持 81 种编程语言语法高亮，内置代码片段模板，开发者友好。

🎵

多媒体嵌入

音频（WaveSurfer.js 波形可视化 + EQ 均衡器）、视频、幻灯片放映，全方位多媒体支持。

🤖

AI 对话集成

支持 Chat/Agent 双模式，多模型切换，工具调用（项目上下文感知），让 AI 成为你创作助手。

🎨

绘图 & 覆盖层

手绘笔/铅笔/荧光笔/彩虹笔，支持撤销重做；图片/形状/文本框/图表自由放置，可拖拽缩放旋转。

🔄

视图切换系统

3D 星图 / 2D 思维导图无缝切换，图层管理，满足不同思考和表达场景的需求。

价值与意义

⚡

效率提升

将「记录 → 整理 → 创作」三个环节统一到同一个工具中，消除工具切换带来的认知负担。3D 可视化帮助用户快速定位知识盲区和关联缺口，大幅缩短从想法到成品的路径。

🧩

思维范式革新

突破传统文档的线性限制，提供「先宏观后微观」的新型知识组织方式。用户可以先在 3D 星图中把握全局，再逐个深入节点细节，更符合人类大脑的发散性思维习惯。

🌍

社会价值

为教育领域提供可视化的知识管理方案，帮助学生建立系统性思维框架。降低专业内容创作的门槛，让非技术人员也能轻松制作包含代码、公式、多媒体的复合型文档。
