使用帮助

底部任务栏(按照从左到右的顺序)

  • 左下角AstroKnot菜单，点击展开显示当前状态所有的网络项目和系统内所有的快速笔记
    • 全局搜索栏：输入节点名称/节点笔记关键字，弹出下拉选项框，点击后会自动聚焦匹配的节点，并打开对应节点文本编辑器并聚焦到关键字所在位置，
    • 网络项目：当前打开的网络项目，标题框的按钮从左到右依次为:批量保存网络项目列表内所有的项目，加载项目，导入markdown文件，新建项目类。 点击列表的项目可切换到其他对应的项目，右键支持重命名，复制，删除项目
    • 快速笔记：系统所有快速笔记，“+”为新建快速笔记，点击打开对应的快速笔记，右键支持重命名，复制，删除笔记
    • 设置（左下角的齿轮）：设置3D场景参数，2D网格背景颜色，主题色，文件保存位置
    • 关闭应用（右下角的电源符号）
  • 节点搜索栏，输入节点名字可弹出上拉选项框，点击节点选项后相机聚焦该节点
  • 全屏/窗口化切换按钮，快捷键：F11
  • 2/3D模式切换按钮
  • Auto：自动布局按钮，点击后会自动布局所有节点
    • 自动排列：自动布局当前选中图层的所有节点
    • 所有图层自动排列：自动布局所有图层图层的所有节点
    • 默认排列：3D模式下，随机布局所有节点
    • 2D模式排列：3D模式下，按照2d模式的布局排列节点，不同图层的节点会按照图层管理列表的由上到下的顺序在3d模式中由下到上堆叠排列
  • 机器人按钮：AI对话框（目前还在开发...）
  • 蓝色向上箭头按钮：里面包含放大/缩小视角按钮和连线标签显示隐藏切换按钮
  • 天气按钮：显示当前的天气信息，悬浮则弹出详细的天气信息，左键单击刷新天气信息，右键弹出选项框可重新定位和手动定位(弹出地址输入栏弹出框)
  • 日期时间：悬浮可显示农历和时辰

右键空白区域菜单

  • 新建根节点/新建步骤节点（随后在2D模式解释二者区别）
  • 撤销/重做对节点连线的所有操作（Ctrl+Z / Ctrl+Y）
  • 粘贴节点：粘贴在复制的所有节点

    • 3D 模式操作

    • 鼠标左键长按拖拽空白区域 → 旋转视角
    • 鼠标滚轮 → 缩放视距
    • WASD / 空格 / ctrl → 平移视角（3D模式）
    • 左键双击节点，进入该节点的文本编辑器
    • 右键节点 → 打开该节点的属性菜单

        • 节点管理

      • 点击节点名称旁边的铅笔符号，对节点重命名，enter回车确认
      • 通过滑动条调整节点大小，圆环转动速度
      • 可设置节点球体固定颜色，点击清除固定颜色，球体在3D华丽模式下恢复到色相变换，极简模式下默认为浅蓝色
      • 新建子节点：点击"新建子节点"按钮，在当前节点下新建一个子节点
      • 添加下一步节点：点击"新建下一步节点"按钮，在当前节点下新建一个下一步节点（随后在2D模式介绍中说明两种节点的关系）

        • 连线管理

        • "添加连线"→ 左键点击目标节点完成连线，两个节点之间支持多条连线
        • "删除连线"→ 左键点击目标节点，多条连线时可多选删除，或者点击对应连线弹出连线标签框，点击“删除此连线”
        • 右键取消 / Esc 退出连线模式
        • 树形连线由父子关系自动生成
      • "折叠/展开子节点"按钮可显示/隐藏该节点的所有后代节点
      • 定位到2/3D视图：可自动切换到2d模式该节点所在图层（或者切换到3D模式），并聚焦到该节点所在位置
      • 移动节点：进入移动模式，可拖拽移动所有节点的位置，点击"确定"按钮完成移动，点击"取消"按钮取消移动
      • 复制节点：复制该节点及其所有子孙节点，右键单击空白区域，弹出选项框，点击“粘贴节点”即完成复制粘贴操作
      • 删除该节点：删除该节点及其所有子孙节点

    2D 模式操作

  • 点击底部"2D模式"按钮切换
  • 滚轮 → 缩放
  • 右键空白区域菜单和节点的属性菜单同3D模式
  • 支持直接拖拽移动节点
  • 上下左右平移移动是WASD/↑↓←→，拖拽空白区域为框选多个节点，或者按住ctrl选择多个节点
    • 框选节点后，可批量对框选的节点调整大小，圆环转速，节点颜色，2/3D形状，设置节点对齐方式（只框主一个节点则不显示）以及移动、复制和删除
    • 框选节点后，可建立群组，即生成一个和框选区域位置大小相同的与节点绑定的半透明边框，

        步骤节点和一般节点的区别

      • 一般子节点是从根节点开始的从左到右的树状图，一般用于思维导图，
      • 下一步节点是从根节点开始的从上到下的树状图，一般用于流程图，
      • 本软件支持同时存在两种类型的节点，并且混合使用，即步骤节点可有从左到右的后代节点，一般节点可有从上到下的后代节点

        多图层管理

      • 2D 模式和 3D 模式均支持多个图层，每个图层独立存放节点，方便对复杂知识图谱按主题、模块或阶段进行分层组织
      • 打开图层面板：点击底部任务栏中的 🗂️ 图层管理按钮（位于任务栏左侧），弹出图层管理浮动面板；面板可拖拽移动位置
      • 创建新图层：在图层面板顶部点击 "＋ 新建图层" 按钮，自动创建并命名为"图层 N"（N 为当前图层数+1）；也可右键某图层选择"在此图层下新建"在其下方插入新图层
      • 切换图层：在图层面板列表中左键点击目标图层名称即可切换；当前激活的图层以高亮样式显示（带 active 标记）；切换后 2D 视图自动刷新仅显示当前图层内的节点，3D 模式下相机将平滑动画移动到该图层平面并显示蓝色高亮矩形
      • 重命名图层：在图层面板中右键点击图层项 → 选择"重命名"，图层名称变为可编辑输入框，修改后按回车或点击空白处确认
      • 删除图层：在图层面板中右键点击图层项 → 选择"删除"，弹出确认对话框；删除后该图层中的所有节点会自动迁移到剩余图层（不会丢失数据）；系统至少保留一个图层不可删除
      • 拖拽排序：在图层面板列表中直接拖拽图层项上下移动，可调整图层顺序；3D 模式下图层顺序对应空间中的上下堆叠关系
      • 图层总览：点击图层面板顶部的 📑 总览按钮，进入全屏 3D 层叠视图，以卡片形式展示所有图层及其节点数量分布；点击任意卡片或标签可快速跳转到对应图层
      • 跨图层复制粘贴：在源图层中选中一个或多个节点（左键单击选中，Ctrl+左键多选）→ 按 Ctrl+C 复制 → 在图层面板中切换到目标图层 → 按 Ctrl+V 粘贴；粘贴的节点会出现在鼠标光标附近（2D 视图）或偏移固定距离（3D 视图），并自动归属到当前图层

        跨图层连线

      • 不同图层之间的节点可以建立连接关系（称为"跨图层连线"或 CrossEdge），形成跨层的知识关联网络。连线数据存储在 crossEdges 数组中，与普通树形连线（父子关系）相互独立
      • 添加跨图层连线 — 方式一（快捷连线）：右键点击源节点 → 选择菜单中的 "添加连线" → 进入连线模式（鼠标变为十字准星）→ 直接左键单击目标节点（无论目标节点是否在当前图层）→ 弹出标签输入框 → 输入连线名称后点"确定"完成
      • 添加跨图层连线 — 方式二（锚点自由绘制）：进入连线模式后，节点的四边中点和四角会出现 8 个蓝色锚点圆圈 → 点击源节点的某个锚点作为起点 → 进入自由绘制状态 → 可在空白处左键点击添加拐点（折线连线）→ 最终点击目标节点的锚点完成连线 → 输入标签确认。此方式可绘制带拐点的复杂折线路径
      • 查看跨图层连线：
        • 当连线的两端节点都在当前图层时，2D 视图中以虚线（dash pattern [6,4]）显示，颜色为全局呼吸色相循环（约30秒转一圈）；若连线端点是步骤连接则显示为紫色(#AA44FF)，一般连接则为青色(#00ccff)
        • 当连线的一端不在当前图层时，该连线在当前视图中不显示（被图层过滤隐藏）
        • 涉及跨图层连线的节点，其外围会绘制一层双实线边框（带呼吸间距动画），直观标识该节点存在跨层关联
      • 跨图层导航：右键点击任一节点时，如果该节点存在跨图层连线，右键菜单中会显示 "跨图层连接" 区域，列出所有跨层关联的节点，格式为 "节点名 (所在图层名)" + 连线标签 → 左键点击其中一项可自动跳转到对方节点所在的图层并将视图聚焦到该节点
      • 删除跨图层连线：右键点击有连线的节点 → 选择 "删除连线" → 进入删除模式 → 点击要断开的目标节点即可移除该条连线；或在跨图层连接区域右键具体连线项 → 子菜单中选择删除
      • 呼吸光效提示：
        • 选中节点且存在跨图层/同层连线时：节点外围叠加双层呼吸光晕（青色 #33ffff/#88ffff 用于一般连接，紫色 #CC66FF/#FF88FF 用于步骤连接），即使另一端节点在其他图层不可见也能感知连接状态
        • 未选中但有关联连线的节点：填充色变为深色调（深蓝 #2a4a5a 或深紫 #2a1a3a），边框变为青色或紫色，带 shadowBlur 呼吸发光效果
        • 跨图层连线本身也带有闪烁光晕效果（glow），当连线端点处于选中状态时光晕增强
      • 取消连线操作：进入连线模式后，按 Esc 键 或右键空白区域可随时取消当前连线操作

    富文本笔记

    • 打开方式：右键点击节点 → 选择"编辑内容"，或双击节点打开富文本编辑器
    • 字体格式：支持加粗、斜体、下划线、删除线、着重号、上标/下标；可设置文字颜色和背景色；支持中英文字体切换（cnfontfamily / enfontfamily）及字号调节（fontplus / fontminus）；支持繁简转换（fan2jian / jian2fan）、拼音标注
    • 段落排版：支持标题级别、行高（lineheight）、段落间距（paraspacing）、对齐方式（左/中/右/两端/分散对齐）、首字下沉（dropcap）、引用块（blockquote）、水平分割线（hr）；支持有序/无序列表缩进调整（outdent / indent）
    • 插入元素
      • 图片：支持从本地选择图片插入，插入后可拖拽缩放、调整透明度和对齐方式（左/居中/右），右键图片可进行格式设置
      • 表格：通过网格选择器快速创建表格，支持行列合并等操作
      • 链接：插入超链接，支持外部 URL 和本地文件路径
      • LaTeX 公式：点击 Σ 按钮打开公式编辑器，支持数学公式渲染
      • 代码块：点击 # 按钮插入代码块，支持语法高亮显示
      • Excel 表格：点击 Excel 按钮嵌入在线 Excel 表格（需网络连接）
      • 图表：点击 Chart 按钮插入 ECharts 图表（折线图/柱状图/饼图等）
      • PPT 幻灯片：点击 PPT 按钮嵌入演示文稿预览
      • 音频/视频：支持插入本地或在线音视频文件
      • 其他：特殊字符（charmap）、表情符号（emoticons）、日期时间（datetime）、分页符（pagebreak）
    • 绘图功能：工具栏提供画笔（drawPen）、铅笔（drawPencil）、荧光笔（drawHighlighter）、橡皮擦（drawEraser）、彩虹笔（drawRainbow），可调节颜色和线条粗细，支持撤销/重做/清空/保存绘图
    • 图形与形状：支持插入形状（shapeformatInsertShape）、文本框（shapeformatInsertTextbox），可设置填充色、边框、阴影、透明度、旋转角度、层级关系；支持艺术字（WordArt）效果
    • 覆盖层块：通过 insertOverlayBlock 插入浮动内容块，可在笔记上层叠加展示额外信息
    • 视图模式：支持页面视图（viewPageBtn）、网页视图（viewWebBtn）、暗黑模式（viewDarkMode）三种查看模式；可全屏编辑（fullscreen）
    • 搜索与替换：内置 searchreplace 功能，支持在长文档中查找和替换文本
    • 目录导航：自动根据标题生成目录（TOC），方便长文档跳转浏览
    • 实时统计：底部实时显示当前字数统计
    • 自动保存：编辑内容自动保存到节点数据中，关闭编辑器后自动持久化
    • 快捷键：Ctrl+B 加粗 / Ctrl+I 斜体 / Ctrl+U 下划线 / Ctrl+S 保存 / Ctrl+F 查找
